Базовый интерпритатор языка Redcode, поддерживающий прямую, косвенную, постинкрементную и предикрементную адресацию.

Список поддерживаемых команд:
MOV
ADD
JMP
JMZ
MUL
DIV

Комментарии в файле программы следуют после символа ';'. 
Не поддерживаются адреса, указывающие за предел поля. Пример: MOV 0, 101
